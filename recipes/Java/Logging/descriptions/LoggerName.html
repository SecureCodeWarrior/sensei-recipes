The name for the logger should reflect the class name.


According to the Oracle documentation, the logger name should be a dot-separated name and should normally be based on the package name or class name of the subsystem.


Before 
public class MyClass {
  Logger logger = Logger.getLogger("Some logger");
    ...
}

After 
public class MyClass {
    Logger logger = Logger.getLogger(MyClass.class.getName());
    ...
}
