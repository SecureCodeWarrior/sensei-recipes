Whenever a test is disabled, e.g. using the @Disabled annotation, a reason for disabling the test should be added. The @Disabled annotation supports this by providing an argument.



Before 
@Disabled
void thisTestIsDisabled() {
  ...
}

After 
@Disabled("TODO: Add a reason for disabling the test")
void thisTestIsDisabled() {
  ...
}
