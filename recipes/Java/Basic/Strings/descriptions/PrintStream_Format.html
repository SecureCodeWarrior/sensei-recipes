Untrusted inputs in format strings should be filtered to prevent injection attacks.


Instead of concatenating untrusted inputs, these should be provided as additional parameters to the format method. This way, they are considered to be inputs instead of parts of the format string.


Before 
System.out.format(untrustedStr + " did not match!");

After 
System.out.format("%s did not match!", untrustedStr);
