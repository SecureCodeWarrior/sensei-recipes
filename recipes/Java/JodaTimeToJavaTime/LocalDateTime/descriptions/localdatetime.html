Migrate from org.joda.time.LocalDateTime to Java Time


Joda-Time LocalDateTime migrates to Java Time LocalDateTime.

It's the same concept, a date and time without time-zone.


Before 
    LocalDateTime now1 = LocalDateTime.now();
After 
     java.time.LocalDateTime now1 = java.time.LocalDateTime.now();
