Migrate from org.joda.time.Period to Java Time


Joda-Time Period migrates to Java Time Period and/or Duration.

It's a similar concept, it represents an amount of time

Joda-Time Period includes years to milliseconds, but java.time only year/month/day (see also (ThreeTen-Extra PeriodDuration).


Before 
    Period period = new Period(periodLong);
After 
     java.time.Period period = java.time.Period.from(periodLong);
