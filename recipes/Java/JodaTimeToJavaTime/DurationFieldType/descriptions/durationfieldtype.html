Migrate from org.joda.time.DurationFieldType to Java Time


Joda-Time DurationFieldType migrates to java.time ChronoUnit.

ChronoUnit has a similar concept but a very different API and implementation.


Before 
    DurationFieldType ft1 = DurationFieldType.centuries();
After 
     ChronoUnit ft1 = ChronoUnit.CENTURIES;
