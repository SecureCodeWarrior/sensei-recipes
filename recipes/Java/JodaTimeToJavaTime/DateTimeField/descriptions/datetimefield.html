Migrate from org.joda.time.DateTimeField to Java Time


Joda-Time DateTimeField and DateTimeFieldType migrate to java.time ChronoField.

ChronoField has a similar concept, although it has a different API and implementation.


Before 
    DateTimeField dateTimeField;
    DateTimeFieldType clockHour = DateTimeFieldType.clockhourOfDay();
After 
    ChronoField dateTimeField;
    ChronoField clockHour = ChronoField.CLOCK_HOUR_OF_DAY;
