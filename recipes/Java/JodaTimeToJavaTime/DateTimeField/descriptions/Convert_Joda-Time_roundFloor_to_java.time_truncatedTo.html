This recipe provides an equivalent for Joda-Time roundFloor, however please note the following precautions:

  • It is assumed that the originating Joda-Time Field was obtained using ISOChronology.getInstanceUTC()
  • This recipe has only been tested to be equivalent based on the following mappings
    • DateTimeFieldType.dayOfMonth() -> ChronoField.DAY_OF_MONTH
    • DateTimeFieldType.dayOfWeek() -> ChronoField.DAY_OF_WEEK
    • DateTimeFieldType.dayOfYear() -> ChronoField.DAY_OF_YEAR
    • DateTimeFieldType.hourOfDay() -> ChronoField.HOUR_OF_DAY
    • DateTimeFieldType.millisOfDay() -> ChronoField.MILLI_OF_DAY
    • DateTimeFieldType.millisOfSecond() -> ChronoField.MILLI_OF_SECOND
    • DateTimeFieldType.minuteOfHour() -> ChronoField.MINUTE_OF_HOUR
    • DateTimeFieldType.secondOfDay() -> ChronoField.SECOND_OF_DAY
    • DateTimeFieldType.secondOfMinute() -> ChronoField.SECOND_OF_MINUTE
