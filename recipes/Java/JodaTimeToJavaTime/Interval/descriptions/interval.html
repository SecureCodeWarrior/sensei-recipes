Migrate from org.joda.time.Interval to Java Time


Joda-Time Interval cannot be migrated to Java Time API.

It's necessary a third-party library. We recommend to use ThreeTen-Extra

ThreeTen-Extra provides additional date-time classes that complement those in Java SE 8.


Before 
    Interval interval = new Interval(start, finish);
After 
     org.threeten.extra.Interval interval = org.threeten.extra.Interval.of(Instant.ofEpochMilli(start), Instant.ofEpochMilli(finish));
