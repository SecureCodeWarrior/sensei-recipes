Migrate from org.joda.time.LocalDate to Java Time


Joda-Time LocalDate migrates to Java Time LocalDate.

It's the same concept, a date without time or time-zone.


Before 
    LocalDate LocalDate1 = new LocalDate();
After 
     java.time.LocalDate LocalDate1 = java.time.LocalDate.now();
