Migrate from org.joda.time.DateTimeFormat to Java Time


Joda-Time DateTimeFormat and ISODateTimeFormat migrate to java.time DateTimeFormatter.

Concepts exposed as static methods on DateTimeFormatter.


Before 
    DateTimeFormatter forPattern = DateTimeFormat.forPattern("somepattern");
    DateTimeFormatter shortDate = DateTimeFormat.shortDate();
After 
    java.time.format.DateTimeFormatter forPattern = java.time.format.DateTimeFormatter.ofPattern("somepattern");
    java.time.format.DateTimeFormatter shortDate = java.time.format.DateTimeFormatter.ofLocalizedDate(FormatStyle.SHORT);
