Migrate from org.joda.time.Duration to Java Time


Joda-Time Duration migrates to java.time Duration.

It's a same concept, amount of time, based on fractional seconds.


Before 
    Duration duration = new Duration(longArg);
    Duration standardDays = Duration.standardDays(longArg);
After 
    java.time.Duration duration = java.time.Duration.ofMillis(longArg);
    java.time.Duration standardDays = java.time.Duration.ofDays(longArg);
