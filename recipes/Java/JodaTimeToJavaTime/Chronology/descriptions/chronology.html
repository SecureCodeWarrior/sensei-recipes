Migrate from org.joda.time.Chronology to java.time.chrono.Chronology


Java Time API has similar concept to Joda-Time in terms of Chronology, although the API and implementation is very different.



Before 
    Chronology c = ISOChronology.getInstance();
    Chronology ic1 = IslamicChronology.getInstance();
    Chronology bc1 = BuddhistChronology.getInstance();
After 
    java.time.chrono.Chronology c = IsoChronology.INSTANCE;
    java.time.chrono.Chronology ic1 = HijrahChronology.INSTANCE;
    java.time.chrono.Chronology bc1 = ThaiBuddhistChronology.INSTANCE;
