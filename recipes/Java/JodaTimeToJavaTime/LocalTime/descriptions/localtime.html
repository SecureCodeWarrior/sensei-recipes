Migrate from org.joda.time.LocalTime to Java Time


Joda-Time LocalDateTime migrates to Java Time LocalTime.

It's the same concept, a time without time-zone.


Before 
    LocalTime now = LocalTime.now();
After 
     java.time.LocalTime now = java.time.LocalTime.now();
