Spring Boot 2.0 deprecated RestTemplateBuilder#requestFactory(ClientHttpRequestFactory), use RestTemplateBuilder#requestFactory(Supplier<ClientHttpRequestFactory> requestFactorySupplier) instead.
Before 
restTemplateBuilder.requestFactory(httpRequestFactory);

After 
restTemplateBuilder.requestFactory(() -> httpRequestFactory);
