setUseRegisteredSuffixPatternMatch(true)allows suffix pattern matching. In other words, a controller mapping to /users, also maps to /users.* (where the wildcard equates to a pattern registered in the contentNegotiationManager). Best practices for content negotation recommend the use of the Accept header over file extensions.
Before:mapping.setUseRegisteredSuffixPatternMatch(true);
After:mapping.setUseRegisteredSuffixPatternMatch(false);
