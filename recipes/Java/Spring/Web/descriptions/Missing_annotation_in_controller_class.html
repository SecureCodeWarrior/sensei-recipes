Controllers and RestControllers should be marked using the respective Spring stereotypes annotations.


Before 
public class MyController {

  @GetMapping("/hello/{name}")
  public String getHello(@PathVariable String name) {
    ...
  }
}
After 
@RestController
public class MyController {

  @GetMapping("/hello/{name}")
  public String getHello(@PathVariable String name) {
    ...
  }
}
